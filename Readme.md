# civic-map-insight

An interactive geospatial analytics project that visualizes Indian public datasets using maps and charts.  
The project focuses on transforming raw CSV and GeoJSON data into intuitive, district- and state-level insights.

Instructions to setup project 
1. Create a virtual environment 
  ``` python -m venv venv```
2. Activate it
   ``venv\Scripts\activate``
3. Install dependencies
   ``` pip install -r requirements.txt ```
4. Run the Streamlit app
   ``` streamlit run app.py ```
